# [kylecain.me](https://kylecain.me)

Personal developer portfolio and blog.

## Tech Stack

- [Astro](https://astro.build) - Static site generator
- [Tailwind CSS](https://tailwindcss.com) - Styling
- [GitHub Pages](https://pages.github.com) - Hosting

## Features

- Modern dark theme with animations
- Animated hero section with typing effect
- Projects fetched from GitHub API
- Blog with content collections
- Responsive design
- GitHub Sponsors integration

## Development

```bash
npm install
npm run dev     # Start dev server at localhost:4321
npm run build   # Build for production
npm run preview # Preview production build
```

## Deployment

Pushes to `master` automatically build and deploy to GitHub Pages via GitHub Actions.
